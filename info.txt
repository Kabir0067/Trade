ҳаминро рости равшан: дар Душанбе (UTC+5) беҳтарин вақт барои скальпинг M1–M15 он соатҳост, ки ликвидият баланд аст (кушодашавии Лондон ва ҳампӯшии Лондон–Ню-Йорк). Барои барномаи ту — дар ҳамин тирезаҳо оғоз кун, то ашёро интихоб кунад ва ордерро худкор боз кунад.

Соатҳои тавсияшаванда (вақти Тоҷикистон, Asia/Dushanbe)
Вақти «тобистона»-и Ғарб (тақр. охири март → охири октябр)

Лондон кушода: 12:00–21:00

Ҳампӯшии Лондон–Ню-Йорк (қавитарин): 17:00–21:00

Осиё→Лондон overlap (FX каме мулоимтар, вале хуб): 12:00–14:00

Вақти «зимистона»-и Ғарб (тақр. охири октябр → охири март)

Лондон кушода: 13:00–22:00

Ҳампӯшии Лондон–Ню-Йорк (қавитарин): 18:00–22:00

Осиё→Лондон overlap: 13:00–14:00

Барои асъорҳо

EURUSD / GBPUSD: равзанаҳои боло — афзалият №1.

XAUUSD (тиллo): ҳамон равзанаҳо; иловатан 22:00-00:00 (тобистон) ё 22:00-01:00 (зимистон) метавонад кор диҳад, аммо танҳо агар спред/ATR хуб бошад.

BTCUSD: 24/7 аст. Агар мехоҳӣ равзана гузорӣ, беҳтарин ҳаракат одатан дар соатҳои ИМА аст:

тобистон: 17:00–01:00; зимистон: 18:00–02:00.
Ё як равзанаи сабуктари субҳона: 05:00–09:00 (бо бозори Токио).

Эзоҳҳо:

Тоҷикистон DST надорад, бинобар ин дар баҳор/тирамоҳ вақтҳои Лондон/Ню-Йорк як соат ба пеш/пас мегарданд (боло аллакай ҷуброн карда шудааст).

Аз соати аввалини душанбе саҳар ва соатҳои охири ҷумъа дурӣ ҷӯй: спредҳо кушода мешаванд.

Агар скрипти аввалатон active_sessions=[(11,15),(16,20)] дошта бошад, он қариб ба тобистон: 12–16, 17–21 меафтад. Барои зимистон стеҷҳои поёнро монед.

Чӣ гузоштан дар коди худ
Агар скрипти «Engine/active_sessions»-ро истифода мебаред

Барои тобистон:

active_sessions=[(12,15),(17,21)]   # 12–15 ва 17–21


Барои зимистон:

active_sessions=[(13,16),(18,22)]   # 13–16 ва 18–22


(Ин соатҳо бе дақиқаанд; агар дақиқа мехоҳед, ба версияи “SymbolPicker” гузаред ва TRADING_WINDOWS истифода баред.)

Агар «SymbolPicker / TRADING_WINDOWS»-ро истифода мебаред

Тобистон (FX/XAU):

TRADING_WINDOWS="12:00-15:30,17:00-21:00"
FX_DAYS="1-5"         # душанбе–ҷумъа
TRADE_TZ="Asia/Dushanbe"


Зимистон (FX/XAU):

TRADING_WINDOWS="13:00-16:30,18:00-22:00"
FX_DAYS="1-5"
TRADE_TZ="Asia/Dushanbe"


BTC (ихтиёрӣ, равзанаи ҷудогона — беҳтар аст процесс-и дуюм):
тобистон:

TRADING_WINDOWS="17:00-01:00"


зимистон:

TRADING_WINDOWS="18:00-02:00"


(ё умуман TRADING_WINDOWS-ро барои BTC холӣ мон, то 24/7 имкон бошад.)

Чӣ тавр бо ENV оғоз кунӣ (Windows PowerShell)
# Мисол: тобистон, FX/XAU
$env:TRADE_TZ="Asia/Dushanbe"
$env:TRADING_WINDOWS="12:00-15:30,17:00-21:00"
$env:FX_DAYS="1-5"
$env:RELAXED="0"
python your_script.py


Ё барои зимистон:

$env:TRADE_TZ="Asia/Dushanbe"
$env:TRADING_WINDOWS="13:00-16:30,18:00-22:00"
$env:FX_DAYS="1-5"
$env:RELAXED="0"
python your_script.py

Хулоса

Барои сигналҳои босифат барномаро дар 17:00–21:00 TJT (тобистон) ё 18:00–22:00 TJT (зимистон) равшан кун — ин қавитарин равзана аст.

Агар мехоҳӣ бештар ҳодиса дошта бошӣ, равзанаи 12/13:00–15/16:30-ро ҳам илова кун.

Барои BTC ё равзанаи иловагӣ процесс-и ҷудогона бо TRADING_WINDOWS-и худ соз.

Агар хоҳӣ, метавонам ба коди ту ивазкунии худкори тобистон/зимистон (бо тақвими DST-и Лондон/Ню-Йорк) илова кунам, то равзанаҳо худ худ иваз шаванд.
